Js-libs

这里推荐的js库是完全主观的，我尽量在每一个领域里给出唯一的，也是我自己的选择

Base

Preprocessor: coffee-script

Client Framework: jquery

Function Lib: underscore

MV* Framework: backbone

Test Framework: jasmine

Template Engine: handlebars

Loader: requirejs

Style Framework: twitter-bootstrap

Options

UI Widgets

  • jquery-ui

    jquery ui组件库

  • jquery-file-upload

    file upload组件

  • select2

    select box组件

  • colorbox

    lightbox组件

  • noty

    notification组件

  • wysihtml5

    Rich Text Editor

UI Components

  • font-awesome

    为bootstrap补充大量的icon和相关行为

  • spin

    可定制的loading指示器spin，无需图像

  • tinycon

    操作favicon，显示bubble或改变图像之类

  • jquery-knob

    不错的环状input

UI Layouts

  • isotype

    实现各种独特的元素布局方式，也可以自定制

HTML5 Polyfills

  • modernizr

    帮助探测识别当前环境对CSS3和HTML5元素的支持

    其他polyfills有很好的列表

Formats

  • jquery-validation

    jquery的form校验库

  • mailcheck

    检查并建议输入的email格式

  • moment

    解析和显示各类时间格式

Misc

  • jquery-pjax

    利用pushstate做到无刷新的页面跳转

  • keymaster

    方便地定义快捷键
