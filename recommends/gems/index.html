Recommend Gems

2012-08-28 11:52:00

这里推荐的Gems是完全主观的，我尽量在每一个领域里给出唯一的，也是我自己的选择

寻找gems最好的地方: ruby toolbox

Rails Test Drive

  • rspec-rails

    行为测试框架rspec的rails集成

  • jasminerice

    javascript测试框架jasmine的rails集成

  • capybara

    web行为模拟框架

  • fabrication

    测试夹具生成

  • ffaker

    最快的假数据生成器

  • database_cleaner

    测试数据库清空工具

  • guard

    监控文件变化的事件并触发任务，有大量扩展，非常适合运行spec

  • guard-rspec

    rspec的guard插件

  • guard-spork

    让guard集成DRb server：spork，预加载测试环境可以节省大量的测试时间

  • guard-jasmine

    jasmine的guard插件，使用phantomjs做无头浏览器测试

  • guard-livereload

    livereload的guard插件，自动刷新浏览器

Rails Auth

  • devise

    用户注册登录，身份验证的整套解决方案

  • omniauth

    实现第三方登录认证，包括Twitter，Facebook，微博等等

  • cancan

    为不同用户设定不同级别的权限

Rails Frontend

  • coffee-rails

    Rails默认使用coffeescript生成javascript

  • jquery-rails

    Rails默认的javascript框架jQuery

  • backbone-rails

    最热门的Javascript MV*框架backbone

  • sass-rails

    Rails默认使用sass代替css

  • compass-rails

    最强大的CSS编程框架Compass，基于sass

  • bootstrap-sass

    CSS样式库Twitter Bootstrap的sass移植

  • handlebars_assets

    最好的js模板引擎handlebars的asset pipeline适配

  • slim

    最清晰和最快的模板引擎

  • kaminari

    最好的分页控制器

  • simpleForm

    比起Rails默认的Form更简洁和更强大的表单控制器

  • client_side_validations

    读取服务端Model的验证逻辑并生成对应的客户端验证逻辑，做到DRY

Rails Backend

  • mongoid

    Mongo ORM

  • nokogiri

    HTML解析器

  • capistrano

    远程部署工具

  • carrierwave

    文件上传管理，支持各类ORM，还可以支持各云平台的存储

  • sunspot

    基于SOLR的全文检索引擎

  • resque

    将任务放入后台队列执行

  • grape

    创建REST API

  • jbuilder

    官方的Json builder

Rails Misc

  • rails_best_practices

    控制代码质量的优秀工具

  • active_admin

    为Rails项目创建后台管理界面，非常强大的可定制UI

  • whenever

    编写和部署Cron

  • backup

    服务器数据备份工具

Ruby General

  • pry

    比起默认的IRB shell更强大的终端调试工具

  • thor

    可以用来代替rake和sake的cli command build工具
